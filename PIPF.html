Definiție parte întreagă

Fie x un număr real.Se numește parte întreagă a lui x cel mai apropiat întreg mai mic sau egal cu x.
Definiția este sugerată de Axioma lui Arhimede: Pentru orice număr real x, există un număr întreg n, unic, astfel încât n ≤ x < n + 1.

Definiție parte fracționară

Fie x un număr real. Se numește parte fracționară a lui x diferența dintre număr și partea lui întreagă.
Partea fracționară a unui număr real este un număr pozitiv subunitar sau nul.
