Definiție putere

Puterea n a unui număr a, notată a n, este o operație între două numere, numite bază, respectiv exponent. În vorbire a n se pronunță "a la puterea n", sau, pe scurt, "a la n". Dacă n este un număr natural, atunci ridicarea la putere poate fi definită ca o înmulțire repetată.

Funcția putere

f:A → B; f(x)=x n; n≥2
  1. n par
    1. f crescătoare pe ℝ + și descrescătoare pe ℝ -
    2. f pară
    3. f nu este injectivă
    4. f este surjectivă

  2. n impar
    1. f crescătoare pe ℝ
    2. f impară
    3. f injectivă
    4. f surjectivă
Funcția putere este inversa funcției radical.
